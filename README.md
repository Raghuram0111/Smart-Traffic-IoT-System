# 🚦 Smart IoT Traffic Monitoring System

## 📌 Description
The Smart IoT Traffic Monitoring System is a web-based application designed to simulate real-time traffic monitoring using IoT concepts. It provides dynamic visualization of traffic data based on selected locations and helps in understanding traffic flow patterns efficiently.

---

## 🎯 Features
- 📍 Location-based traffic monitoring  
- 📊 Real-time data simulation and updates  
- 🌐 Interactive web dashboard  
- 🔄 Dynamic traffic status display  
- 📈 Easy-to-use user interface  

---

## 🧠 Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Concepts:** IoT, Real-Time Data Simulation
